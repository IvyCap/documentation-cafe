= Asciidoc Quick Reference Guide
:toc: right

Asciidoc is a  plain text markup language designed for creating technical content. It is feature rich with senamtic elements and built to be moduler with content reusability. Asciidoc can be read as is or converted in to mutiple formates including html, pdf, e-books, man pages, slide presentation and many more. 

The https://asciidoc.org[Asciidoc] project is maintained by the https://asciidoc-wg.eclipse.org/[Eclipse Foundation]. The full https://docs.asciidoctor.org/asciidoc/latest/[AssciiDoc Specification is found here]
